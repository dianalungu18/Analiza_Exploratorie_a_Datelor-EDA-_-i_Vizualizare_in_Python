# Analiza_Exploratorie_a_Datelor-EDA-_-i_Vizualizare_in_Python
Proiect de analiză a datelor utilizând Python, Pandas și Matplotlib, incluzând curățarea datelor și vizualizări pentru evidențierea insight-urilor.
Acest proiect realizează analiza și preprocesarea unui set de date folosind Python. Sunt utilizate biblioteci precum Pandas pentru manipularea datelor și Matplotlib/Seaborn pentru vizualizare. Proiectul include curățarea datelor, tratarea valorilor lipsă și generarea de grafice pentru interpretarea rezultatelor.
